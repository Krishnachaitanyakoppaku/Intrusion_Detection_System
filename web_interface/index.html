Rule Format Help

For ICMP/Ping rules: Gemini will automatically use alert icmp any any -> any any format
For TCP/UDP rules: Specify port numbers (e.g., HTTP=80, SSH=22, HTTPS=443)
Format: alert [protocol] any any -> any [port] (msg:"message"; priority:1-5);
